## 📊 Dataset Overview
**BWS Maid_Pro** is a high-fidelity, commercial-grade dataset comprised of 16,000+ assets, optimized for Generative AI vision training and high-performance neural network development. This archive is engineered for enterprise-level ingestion, featuring complete metadata enrichment and strict CLEAR Act 2026 compliance.

### 📦 Archive Contents
* **`Maid_ServiceAnonymized_JpeGs.zip`**: The full 12GB commercial payload. All assets have undergone a rigorous biometric-destruction pipeline using MediaPipe Tasks API.
* **`Anonymized_Maid_Pro_Enhanced_Manifest.csv`**: A forensically audited manifest containing MD5 hashes, camera metadata, and licensing attributes for every asset in the set.
* **`/Samples`**: Watermarked visual previews for pipeline evaluation.
